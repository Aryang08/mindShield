# MindShield AI

A Generative AI-powered solution that helps students preparing for high-stakes examinations such as NEET, JEE, CUET, CAT, GATE, UPSC, and Board Exams.

## Problem Statement Alignment

MindShield AI goes beyond simple tracking by:
- **Understanding emotions** through natural language journals using advanced Gemini LLM analysis.
- **Detecting hidden stress patterns** based on historical mood data, study hours, and journal sentiment.
- **Identifying emotional triggers** via keyword extraction and comparative anxiety scaling.
- **Predicting burnout** using predictive generative modeling to warn students before they hit a critical breakdown.

## Features
- AI Journal Analyzer
- Smart Mood Tracking
- Burnout Prediction
- AI Wellness Coach
- Adaptive Mindfulness
- Stress Forecast
